# Copy Trade

A Python application that allows copying trades from one trading account to another.

## Features

- Connect to MetaTrader 5 trading accounts
- Copy trades between accounts
- Support for FundedNext and other brokers

## Requirements

- Python 3.8+
- MetaTrader 5 terminal installed
- `MetaTrader5` Python package

## Installation

```bash
pip install -r requirements.txt
```
